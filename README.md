### Installation

1. Enter folder
```
$ cd auth-web-demo
```

2. Install dependencies
```
~/auth-web-demo$ npm install
```

3. Serve with hot reload at localhost:8000 
```
~/auth-web-demo$ npm run dev
```

4. Build for production with minification.
```
~/auth-web-demo$ npm run build
```

### CheckCode
